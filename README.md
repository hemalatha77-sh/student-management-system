# Student Management System

This is a simple Student Management System web application that allows users to manage student records. Users can add, view, and delete student information using a simple web interface.

## Features
- Add student details
- Display student list
- Delete student records
- Simple and user-friendly interface

## Technologies Used
- HTML
- CSS
- JavaScript

1. Download or clone the repository.
2. Open the project folder.
3. Open the `index.html` file in your browser.
